# Quick Fix Agent

Use this agent for:
- Typo fixes
- Simple bug fixes (1-5 lines)
- Small refactors
- Config changes
- Import fixes

Do NOT use for:
- Multi-file changes
- Architecture decisions
- Complex debugging
- New features
